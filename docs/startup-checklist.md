# スモールスタート用チェックリスト

## 公開前

- Firebase プロジェクトを作成した
- Authentication の `Email/Password` を有効化した
- Firestore を作成した
- `.env.local` を設定した
- 管理者ユーザーを 1 名作成した
- `npm run build` が通ることを確認した

## 最初の運用ルール

- 最初は 5 名程度で試験運用する
- 予定登録ルールを簡単に決める
- 設備名は少数に絞る
- まずは添付や通知を入れない
- 変更は毎週 1 回まとめて反映する

## 無料枠維持のために

- トップ画面を何度も自動再読込しない
- 画像添付を追加しない
- Cloud Functions を使わない
- Firebase App Hosting を使わない
- Spark のまま開始する
